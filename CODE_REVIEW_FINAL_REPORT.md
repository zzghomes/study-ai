**相关文件**:
- `data-21days.js`

### 2. 数据文件加载顺序问题 【已修复】

**问题描述**: `index.html` 中未引入 `data.js`，导致 `AI_PLATFORMS` 变量未定义

**影响**:
- 快捷入口平台卡片无法渲染
- 任务详情中的平台按钮无法显示
- 控制台报错：`AI_PLATFORMS is not defined`

**修复内容**:
- 在 `index.html` 中添加了 `<script src="data.js"></script>`，确保在其他脚本之前加载

**相关文件**:
- `index.html`

---

### 3. 硬编码天数问题 【已修复】

**问题描述**: `app.js` 中 `updateWelcome()` 方法硬编码了 `7` 天

**原代码**:
```javascript
} else if (completedCount < 7) {
  document.getElementById('welcome-message').textContent = `已完成 ${completedCount}/7 天，继续加油！`;
```

**影响**:
- 切换到 21 天模式时显示错误的天数

**修复内容**:
- 使用动态的 `LEARNING_TASKS.length` 替代硬编码值

**修复后代码**:
```javascript
const totalDays = LEARNING_TASKS.length;
if (completedCount === 0) {
  document.getElementById('welcome-message').textContent = '准备好今天的 AI 冒险了吗？';
} else if (completedCount < totalDays) {
  document.getElementById('welcome-message').textContent = `已完成 ${completedCount}/${totalDays} 天，继续加油！`;
```

**相关文件**:
- `app.js`
