## ✨ 设计理念

### 核心原则
1. **高级审美** - 马卡龙/莫兰迪色系，低饱和度高级感
2. **自然融合** - 真实风景图片作为背景
3. **简洁优雅** - 移除多余动画，保留必要交互
4. **视觉舒适** - 柔和配色，长时间使用不疲劳

## 🎨 配色详解

### 马卡龙色系特点
```
饱和度：20-40%（低饱和）
明度：80-95%（高明度）
色相：柔和过渡
对比：低对比度
```

```
质感：高级沉稳
```
