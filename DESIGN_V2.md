## 🔐 家长密码保护

### 功能说明
- **首次使用** - 自动弹出设置密码窗口
- **密码格式** - 4-6 位数字
- **再次访问** - 需要输入正确密码
- **忘记密码** - 可以重置密码

### 使用流程
```
点击"家长"按钮
  ↓
弹出密码输入框
  ↓
输入 4-6 位数字密码
  ↓
验证成功 → 进入家长中心
验证失败 → 提示错误
```

### 设置密码
1. 首次点击家长模块
2. 输入 4-6 位数字
3. 确认密码
4. 保存成功

### 重置密码
1. 点击"忘记密码？"
2. 确认重置
3. 设置新密码

## 🔧 技术实现

### CSS 变量
```css
:root[data-theme="coral"] {
  --bg-primary: #FF6B6B;
  --bg-gradient: linear-gradient(...);
  --accent-primary: #FF6B6B;
  --card-bg: rgba(255,255,255,0.95);
}
```

### 密码保护
```javascript
// 本地存储密码
localStorage.setItem('parentPassword', password);

// 验证密码
const saved = localStorage.getItem('parentPassword');
if (input === saved) {
  // 验证成功
}
```

### 主题切换
```javascript
// 切换主题
document.documentElement.setAttribute('data-theme', themeName);

// 保存偏好
localStorage.setItem('aiExplorerTheme', themeName);
```
