## 🚀 使用指南

### 1. 多孩子管理
1. 进入家长中心
2. 点击"管理孩子"
3. 添加多个孩子，每个孩子有独立的学习数据
4. 支持头像上传（emoji 或照片）
5. 点击孩子卡片切换当前用户

### 2. PDF 报告导出
1. 进入家长中心
2. 点击"导出 PDF 报告"按钮
3. 自动生成包含学习数据、能力雷达图、徽章作品的 PDF
4. PDF 将自动下载

### 3. 学习热力图
1. 进入家长中心
2. 点击"查看热力图"按钮
3. 显示过去 180 天的学习记录
4. 再次点击隐藏热力图

### 4. 能力雷达图
- 自动在家长中心显示
- 5 个维度：逻辑思维、创意表达、信息检索、安全认知、专注力
- 点击"月度对比"查看能力变化
- 点击"个性化推荐"获取提升建议

### 5. 休息提醒
- 默认每 50 分钟自动提醒
- 支持浏览器通知
- 可在设置中调整间隔

## 📝 模块 API

### 休息提醒模块
```javascript
// 启用提醒
app.restReminder.enable();

// 禁用提醒
app.restReminder.disable();

// 设置间隔（分钟）
app.restReminder.setInterval(30);
```

### 多孩子管理
```javascript
// 添加孩子
app.multiChild.addChild({ name: '小明', avatar: { type: 'emoji', emoji: '👦' } });

// 切换孩子
app.multiChild.switchChild(childId);

// 获取当前孩子
const child = app.multiChild.getCurrentChild();

// 显示添加孩子模态框
app.multiChild.showAddChildModal();

// 显示管理孩子列表
app.multiChild.showManageChildrenModal();
```

### PDF 导出
```javascript
// 导出报告
app.exportPDFReport();

// 导出指定孩子的报告
app.pdfExport.exportReport(childId);
```

### 热力图
```javascript
// 初始化
app.initHeatmap();

// 渲染热力图
app.heatmap.render('heatmap-container');

// 刷新热力图
app.heatmap.refresh();
```

### 切换孩子
```javascript
// 切换当前孩子并刷新页面
app.switchChild(childId);
```
