## 2. 产品概述

### 2.1 产品定位
AI 探险家是一款专为 10-14 岁儿童设计的 AI 学习乐园，通过游戏化的方式帮助孩子了解和使用主流 AI 平台，培养 AI 素养。

### 2.2 目标用户
- **主要用户：** 10-14 岁小学生
- **次要用户：** 家长（监督学习进度）
- **使用场景：** 家庭学习、课后辅导、假期学习

### 2.3 产品目标
1. 让孩子了解 7+ 主流 AI 平台的特点和用途
2. 通过 14 天系统化学习掌握 AI 工具的基本使用方法
3. 培养安全使用 AI 的意识和习惯
4. 激发孩子对 AI 技术的兴趣和探索欲

### 2.4 核心价值
- 🎮 **游戏化学习** - 等级、徽章、连击系统激励学习
- 📚 **系统化课程** - 14 天循序渐进的学习计划
- 🛡️ **安全第一** - 强调 AI 使用安全和隐私保护
- 👨‍👩‍👧 **家长参与** - 学习报告和能力评估让家长了解进度
- 📖 **知识拓展** - AI 术语、平台介绍、提示词模板学习

### 3.2 详细功能说明

#### 3.2.1 首页功能

**欢迎卡片**
- 显示视频问候（支持自动播放）
- 根据时间显示不同问候语（早上好/下午好/晚上好）
- 显示学习进度提示

**XP 等级系统**
- 10 个等级：新手→学徒→探索者→冒险家→达人→专家→大师→传奇→创造者→先知
- XP 获取：完成学习任务获得
- 等级图标：🌱🌿🧭⚔️🌟💫🏆👑🛠️🔮

**连击计数器**
- 记录连续学习天数
- 每日签到更新
- 间隔超过 1 天重置

**学习进度卡片**
- 总进度百分比
- 已完成天数
- 已完成任务数
- 获得徽章数

**今日任务**
- 显示第一个未完成的任务
- 点击跳转到任务详情
- 全部完成后显示庆祝信息

**快捷入口**
- 显示前 6 个 AI 平台卡片
- 点击跳转到对应平台官网

- 每个任务包含：标题、提示词、小贴士、推荐平台、XP 奖励

#### 3.2.5 知识库模块

**AI 术语词典**
- 14 个 AI 术语解释
- 分类：基础概念、技术概念、应用场景、平台工具

- 小测验功能

**AI 平台百科**

- 标记已读
- 收藏功能
- 平台对比（最多 4 个）
- 排序：按时间/按字母

**提示词模板**
- 11 个提示词模板
- 分类：写作、编程、学习、分析
- 模板详情：标题、提示词内容、使用说明

**学习资源**
- 5 类学习资源：入门指南、写作辅助、编程学习、学习研究、生活应用

- 难度分级：入门、进阶、高级

- 统一收藏页面
- 分类展示：收藏的术语、收藏的平台、收藏的模板

**平台对比**
- 支持 2-4 个平台对比

- 对比表格展示

**用户贡献**

- 填写名称、描述、举例等信息

#### 3.2.6 成长报告（原家长中心）

- 开始日期

- 完成率

- 基于任务完成情况计算得分
- 可视化雷达图展示

**月度对比**
- 对比上月和本月能力得分

**个性化推荐**
- 基于能力短板推荐学习任务
- 提供改进行动建议

**AI 评语**
- 自动生成学习评语
- 包含优势表扬和改进建议
- 支持重新生成

**设置功能**
- 修改孩子姓名
- 重置进度
- 导出数据

## 4. 非功能需求

### 4.1 性能要求
- 页面加载时间 < 3 秒
- 操作响应时间 < 500 毫秒
- 支持离线使用

### 4.2 安全要求
- 数据本地存储，不上传服务器
- 不收集儿童个人信息
- 无密码保护（简化访问流程）

### 4.3 兼容性要求
- 浏览器：Chrome、Edge、Safari 最新版
- 设备：桌面电脑、平板、手机
- 系统：Windows、macOS、iOS、Android

### 4.4 可用性要求
- 界面简洁，适合儿童使用
- 文字大小适中，易于阅读
- 操作反馈清晰

```javascript
APP_DATA = {
  childName: string,        // 孩子姓名
  startDate: timestamp,     // 开始日期
  lastActive: timestamp,    // 最后活跃时间

  currentAbilityScores: object,  // 当前能力得分
  lastAbilityUpdate: timestamp   // 能力数据更新时间
}
```

### 5.2 任务数据结构

```javascript
LEARNING_TASKS = [{
  day: number,              // 第几天
  title: string,            // 任务标题
  platform: string,         // 平台名称
  platformId: string,       // 平台 ID
  difficulty: number,       // 难度 (1-5)
  completed: boolean,       // 是否完成
  xp: number,               // 经验值
  cases: array,             // 实践任务列表
  badge: string             // 关联徽章 ID
}]
```
