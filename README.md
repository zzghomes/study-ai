# 🤖 AI 探险家 - 少儿 AI 学习乐园

帮助 12 岁孩子了解和学习使用主流 AI 平台的互动学习应用。

## 📱 功能特点

- ✅ **7 天学习计划** - 系统化学习 7 大主流 AI 平台
- ✅ **进度追踪** - 实时查看学习进度和完成情况
- ✅ **成就系统** - 9 个徽章激励孩子完成学习
- ✅ **奖励机制** - 4 个可解锁的家庭奖励
- ✅ **作品墙** - 展示孩子的 AI 创作作品
- ✅ **家长中心** - 查看学习报告和使用指南
- ✅ **离线使用** - 数据保存在本地，无需服务器
- ✅ **PWA 支持** - 可添加到手机主屏幕

## 🚀 快速开始

### 方法 1：直接打开（最简单）

1. 找到 `index.html` 文件
2. 双击用浏览器打开（推荐 Chrome、Edge、Safari）
3. 开始使用！

### 方法 2：本地服务器（推荐）

```bash
# 如果有 Python
cd ai-explorer
python -m http.server 8080

# 然后用浏览器打开 http://localhost:8080
```

### 方法 3：部署到网络（免费）

#### GitHub Pages
1. 创建 GitHub 账号
2. 创建新仓库（如 `ai-explorer`）
3. 上传所有文件
4. 在 Settings → Pages 启用 GitHub Pages
5. 访问 `https://你的用户名.github.io/ai-explorer`

#### Vercel（推荐）
1. 访问 https://vercel.com
2. 用 GitHub 账号登录
3. 点击 "New Project"
4. 选择你的仓库
5. 点击 "Deploy"
6. 获得免费域名 `https://ai-explorer.vercel.app`

#### Netlify
1. 访问 https://netlify.com
2. 拖拽 `ai-explorer` 文件夹到部署区域
3. 获得免费域名

## 📱 添加到手机主屏幕

### iPhone/iPad (Safari)
1. 用 Safari 打开应用
2. 点击底部 "分享" 按钮
3. 选择 "添加到主屏幕"
4. 点击 "添加"

### Android (Chrome)
1. 用 Chrome 打开应用
2. 点击右上角菜单（三个点）
3. 选择 "添加到主屏幕"
4. 点击 "添加"

## 🎓 使用建议

### 给家长
1. **陪伴学习** - 建议每天陪伴孩子完成任务，讨论 AI 的回答
2. **引导思考** - 鼓励孩子思考 AI 回答的优缺点
3. **安全第一** - 强调不向 AI 透露个人信息
4. **控制时间** - 建议每天使用不超过 30 分钟
5. **鼓励分享** - 让孩子把作品分享给家人朋友

### 给孩子
1. **每天一任务** - 完成当天的任务再玩下一个
2. **多尝试** - 同一个问题可以问不同的 AI
3. **记笔记** - 把你的发现记录下来
4. **多思考** - AI 说的不一定都对，要学会判断
5. **守规则** - 遵守和爸爸妈妈制定的 AI 使用规则

## 🔧 自定义修改

### 修改孩子姓名
在家长中心页面直接修改，或编辑 `data.js` 中的：
```javascript
const APP_DATA = {
  childName: '你的孩子姓名',
  // ...
};
```

### 修改奖励内容
编辑 `data.js` 中的 `REWARDS` 数组，自定义奖励内容。

### 调整任务难度
编辑 `data.js` 中的 `LEARNING_TASKS` 数组，修改每个任务的 `difficulty`（1-3）。

### 添加新的 AI 平台
编辑 `data.js` 中的 `AI_PLATFORMS` 数组，添加新平台信息。

## 📊 数据说明

- **数据存储**：所有数据保存在浏览器本地存储（localStorage）
- **数据导出**：在家长中心可以导出 JSON 格式的学习数据
- **数据重置**：在家长中心可以重置所有进度
- **隐私安全**：数据只保存在本地，不会上传到任何服务器

## ⚠️ 使用须知

1. **年龄建议**：推荐 10-14 岁儿童使用
2. **家长监督**：建议在家长指导下使用
3. **平台注册**：部分 AI 平台可能需要家长协助注册账号
4. **使用时间**：建议每天使用不超过 30 分钟
5. **信息安全**：不要向 AI 透露姓名、学校、地址等个人信息

## 🆘 常见问题

**Q: 打不开页面怎么办？**
A: 尝试用 Chrome、Edge 或 Safari 浏览器，确保是最新版本。

**Q: 数据丢失了怎么办？**
A: 数据保存在浏览器中，清除浏览器数据会丢失。建议定期在家长中心导出数据。

**Q: 可以在多个设备使用吗？**
A: 可以，但每个设备的数据是独立的，不会同步。

**Q: AI 平台需要付费吗？**
A: 本应用列出的 AI 平台都有免费版本，基础功能足够学习使用。

**Q: 如何打印学习证书？**
A: 完成全部学习后，在家长中心可以看到完成报告，可以用浏览器的打印功能打印。
