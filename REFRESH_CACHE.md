# 🔄 强制刷新浏览器缓存

**重要：** 粘土主题样式已更新，但浏览器可能缓存了旧版本。

---

## ⚡ 快速刷新（推荐）

### Windows:
1. 按 `Ctrl + Shift + Delete`
2. 勾选"缓存的图片和文件"
3. 点击"清除数据"
4. 刷新页面（F5）

### 或更简单:
- 按 `Ctrl + F5` (强制刷新)
- 或 `Ctrl + Shift + R`

### Mac:
- 按 `Cmd + Shift + R`

---

## 🧪 验证粘土效果

刷新后，检查以下几点：

### 1. 进度条颜色
- ✅ **应该是珊瑚橙色渐变**（红橙→浅橙→黄）
- ❌ 不是蓝紫色渐变

### 2. 卡片阴影
- ✅ **底部有明显阴影**（卡片看起来浮起）
- ✅ **左上角有白色高光**
- ❌ 不是平的或只有淡阴影

### 3. 圆角
- ✅ **非常大**（32px）
- ❌ 不是小圆角或直角

---

## 🔍 在浏览器中检查

### Chrome/Edge:
1. 按 `F12` 打开开发者工具
2. 点击右上角的"选择元素"图标（箭头）
3. 点击页面上的进度条
4. 在右侧"Styles"面板查看应用的样式

**应该看到：**
```css
background: linear-gradient(90deg, #FF6B5B, #FFA06B, #FFB347) !important;
```

### 或者在控制台执行:
```javascript
const fill = document.querySelector('.progress-fill');
console.log('进度条背景:', getComputedStyle(fill).background);
```

**应该输出：**
```
rgb(255, 107, 91) 或 #FF6B5B (珊瑚橙色)
```

**如果输出蓝色/紫色：** 缓存未清除，请重复上面的刷新步骤。

---

## 🆘 还是蓝紫色？

### 方法 1: 完全关闭浏览器
1. 完全关闭浏览器（所有窗口）
2. 重新打开
3. 访问页面
4. 按 `Ctrl + F5`

### 方法 2: 使用无痕模式
1. 按 `Ctrl + Shift + N` (Chrome) 或 `Ctrl + Shift + P` (Firefox)
2. 在新窗口中打开 `index.html`
3. 检查进度条颜色

### 方法 3: 手动应用样式（临时）
在控制台执行：
```javascript
document.querySelectorAll('.progress-fill, .xp-fill').forEach(el => {
  el.style.background = 'linear-gradient(90deg, #FF6B5B, #FFA06B, #FFB347)';
});

document.querySelectorAll('.hero-section, .progress-card').forEach(el => {
  el.style.boxShadow = '12px 12px 24px rgba(180, 140, 120, 0.18), -8px -8px 20px rgba(255, 255, 255, 1)';
});
```
