## 📝 技术说明

### 休息提醒模块 API

```javascript
// 启用提醒
app.restReminder.enable();

// 禁用提醒
app.restReminder.disable();

// 设置间隔（分钟）
app.restReminder.setInterval(30); // 30 分钟

// 手动启动
app.restReminder.start();

// 跳过休息
app.restReminder.skipRest();

// 完成休息
app.restReminder.finishRest();
```
