## 📖 使用指南

### 快速开始
```bash
# 1. 直接在浏览器打开 index.html
# 2. 或者使用本地服务器
python -m http.server 8000

# 访问 http://localhost:8000
```

### 添加新用户
1. 进入家长中心
2. 点击"添加孩子"
3. 填写信息并上传头像
4. 切换到新账号开始学习

### 导出报告
1. 完成至少 1 天学习
2. 进入家长中心
3. 点击"导出 PDF 报告"
4. 下载并分享
